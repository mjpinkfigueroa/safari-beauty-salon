# Safari — Beauty Salon Website

A full-stack web application for a local beauty salon, featuring a service menu,
online appointment booking system, and an admin dashboard for the salon owner.

Built as a portfolio project using agile development practices.

---

## 🛠️ Tech Stack

| Layer      | Technology          |
|------------|---------------------|
| Frontend   | HTML, CSS, JavaScript |
| Backend    | Python (Flask)      |
| Database   | SQLite (dev) → PostgreSQL (prod) |
| Deployment | Netlify (frontend) + Render (backend) |
| Version Control | Git + GitHub   |

---

## ✨ Features

- Responsive multi-page website
- Service menu with pricing
- Appointment booking system with real-time availability
- Email confirmation on booking
- Admin dashboard for the salon owner
- Mobile-friendly design
